¿Que es HTML y para que sirve?
Hyper Text Markup Lenguaje, este es un lenguaje de Marcado que sirve para
estructurar el contenido de un pagina web
¿Cuàl es la diferencia entre HTML,CSS, JavaScript?
HTML: Estructura de la Pàgina Web
CSS: Este proporciona los estilos o colores a la Pàgina
JavaScript: Con este agregamos la fucnionalidad o lo que hace la pàgina web
¿Por que HTML no es un lenguaje de programaciòn?
Por es un tipo de lenguaje de Mrcado o dicho de otro modo no podemos realizar operciones logicas 
con este.
